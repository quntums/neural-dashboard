# Neural Dashboard

Live Svelte 5 neural network visualization built with runes, Vite, and Tailwind CSS.

**Live demo:** (add Vercel link after deploy)

## Stack
- Svelte 5 ($state runes)
- Vite 8.0.16
- Tailwind CSS 4.x
- Canvas API for real-time graph

## Features
- Live metrics: Activity, Loss, Throughput, Neurons, Synapses, Epoch
- 60-node neural graph with pulsing glow, distance-based links
- Dark lab theme with glass panels
- 60fps simulation loop (no backend)

## Run locally
npm install
npm run dev

## Build
npm run build
